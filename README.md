# Delicius Page

## Este pagina foi feita utilizando

- React
- Bootstrap
- JQuery

## Como utilizar?

- Clone o repositório:
  `git clone https://github.com/crislerwin/delicius-react-app.git`

- Entre na pasta
  ` cd delicius-react-app`

- Instale as dependencias
  `npm install`

- Rode a aplicação e acesse http://localhost:3000
  `npm start`
